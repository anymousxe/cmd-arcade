## What is this?

**CMD Arcade** is a simple open-source terminal app that lets you play **9 different minigames** from a menu, all inside your command prompt / terminal.

- ✅ One file: `cmd_arcade.py`
- ✅ Standard library only (no installs needed)
- ✅ Works on Windows / macOS / Linux
- ✅ Includes a **minimax AI** in Tic‑Tac‑Toe (that’s the “nerdy flex”)

---

## Games included

1. Guess the Number  
2. Rock Paper Scissors  
3. Hangman  
4. Tic‑Tac‑Toe (**minimax AI**)  
5. Blackjack  
6. Word Scramble  
7. Minesweeper (small grid)  
8. Quiz Bowl  
9. Bulls and Cows
