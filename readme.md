# CMD Arcade

A small open-source terminal arcade featuring **9 minigames** you can play in your command prompt.

- No external dependencies (Python standard library only)
- Works on Windows/macOS/Linux
- Clean menu launcher
- Includes a minimax AI in Tic-Tac-Toe

## Games included

1. Guess the Number  
2. Rock Paper Scissors  
3. Hangman  
4. Tic-Tac-Toe (minimax AI)  
5. Blackjack  
6. Word Scramble  
7. Minesweeper (small grid)  
8. Quiz Bowl  
9. Bulls and Cows  

## Requirements

- Python 3.10+

## Run without installing

From the repo root:

```bash
python -m cmd_arcade
